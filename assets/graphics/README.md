# Graphics

Decorative **SVG** elements (florals, sparks, illustrations). Placeholders are
in place; replace or add freely.

| File               | Used for                              |
|--------------------|----------------------------------------|
| `floral.svg`       | Organic flower silhouette accents      |
| `star.svg`         | Cadet-grey spark / sparkle accent      |
| `footer-illus.svg` | Centre illustration in the 3-col footer |

Notes:
- Prefer `fill="currentColor"` so brand colours can be applied via CSS.
- Stripe and scallop dividers are pure CSS (`.divider--stripe`,
  `.divider--scallop`) — no image needed.
