# Logos

Drop the **Nails Lindes** brand logos here as **SVG**. Placeholder versions are
already in place so the site renders — replace them with the real files using
the **same filenames** and everything keeps working.

| File             | Where it's used                         | Notes                              |
|------------------|------------------------------------------|------------------------------------|
| `mark.svg`       | Header brand, footer                     | Small square icon mark (~44×48)    |
| `badge.svg`      | Home hero diamond badge (light)          | Round badge, dark ink on light     |
| `badge-dark.svg` | Reserved for dark backgrounds            | Light ink for use on Dark Coffee   |
| `wordmark.svg`   | Optional horizontal "NAILS LINDES" lockup | Add if you have one                |

Tips:
- Keep colours to the brand palette (Stone Brown `#645A4E`, Tan `#D8C094`…).
- If you want a logo to inherit colour from CSS, set its fills to
  `currentColor` instead of a fixed hex.
- Just tell Claude when you've added a file and it can wire it in / optimise it.
